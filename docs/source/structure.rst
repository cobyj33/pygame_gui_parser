Structure
=======================

The basic structure of a pygame_gui_xml document is as follows

.. code-block::
    :linenos:

    <?xml version="1.0" encoding="UTF-8"?>
    <!DOCTYPE pygamegui>

    <pygamegui> 
        <head>

        </head>

        <body>
            
        </body>
    </pygamegui>
